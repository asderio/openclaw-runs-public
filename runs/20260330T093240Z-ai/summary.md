# 작업 요약

인문 분야 학습자에게 AI를 어떻게 가르칠지에 대한 한국어 보고서를 작성했다. 보고서에는 교육 원칙, 6회차 커리큘럼, `NotebookLM` 활용 수업 방식, 과제 예시, 평가 기준, 현장 적용 시 유의점을 포함했다.

`NotebookLM`을 이용한 직접 조사도 시도했지만, 현재 세션에서는 관련 MCP 호출이 `user cancelled MCP tool call`로 취소되어 특정 노트북 자료를 인용한 맞춤형 보고서는 만들지 못했다. 따라서 이번 산출물은 범용형 실무 보고서이며, 후속으로 실제 노트북 URL이나 노트북 접근이 가능해지면 맞춤형 버전으로 확장할 수 있다.
