# /commands

확인 가능한 슬래시 명령은 아래와 같습니다.

- `/commands` : 사용 가능한 명령/기능 안내를 표시합니다.

이 요청 폴더 안에는 추가 슬래시 명령 목록이나 매니페스트가 없어서, 위 항목만 검증할 수 있었습니다.

대신 이 환경에서는 자연어 요청으로 아래 작업을 처리할 수 있습니다.

- 조사/리서치 요약 작성
- Gmail 메일 검색, 요약, 초안 작성
- Google Calendar 일정 확인 및 일정 잡기
- Google Drive/Docs/Sheets 파일 검색 및 편집
- 브라우저 자동화와 웹 UI 확인
- Stitch 기반 화면/와이어프레임 생성

예시 요청:

- `오늘 일정 요약해줘`
- `지난주 민수한테 온 메일 찾아줘`
- `드라이브에서 분기 계획 문서 찾아줘`
- `NotebookLM로 AI 에이전트 자료 조사해줘`
